### **安装步骤**

1. Clone 仓库并进入仓库目录

```sh
git clone https://github.com/PiDanShouRouZhouXD/Sakura_Launcher_GUI.git
cd Sakura_Launcher_GUI
```

1. 安装依赖

```sh
# 其他依赖: xmake, just, C++工具链
pip install -r requirements.txt
just run
```

### **注意事项**

- 请确保已安装Python3.x环境
- 建议使用包管理器安装依赖，如miniconda。
